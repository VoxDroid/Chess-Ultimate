## 🚀 Erste Schritte

Um Schach Ultimativ lokal zu starten:

1. **Klone das Repository**:
    ```bash
    git clone https://github.com/VoxDroid/Chess-Ultimate.git
    cd Chess-Ultimate
    ```

2. **Öffnen Sie es in Ihrem Browser**:
    ```bash
    open index.html
    ```
