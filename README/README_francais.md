## 🚀 Démarrage

Pour commencer à jouer à Échecs Ultime localement :

1. **Clonez le dépôt** :
    ```bash
    git clone https://github.com/VoxDroid/Chess-Ultimate.git
    cd Chess-Ultimate
    ```

2. **Ouvrez-le dans votre navigateur** :
    ```bash
    open index.html
    ```
