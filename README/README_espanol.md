## 🚀 Comenzando

Para comenzar a jugar Ajedrez Definitivo localmente:

1. **Clona el repositorio**:
    ```bash
    git clone https://github.com/VoxDroid/Chess-Ultimate.git
    cd Chess-Ultimate
    ```

2. **Ábrelo en tu navegador**:
    ```bash
    open index.html
    ```
