## 🚀 시작하기

로컬에서 체스 얼티밋을 시작하려면:

1. **리포지토리 클론**:
    ```bash
    git clone https://github.com/VoxDroid/Chess-Ultimate.git
    cd Chess-Ultimate
    ```

2. **브라우저에서 열기**:
    ```bash
    open index.html
    ```
