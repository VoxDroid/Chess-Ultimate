## 🚀 快速开始

要在本地运行国际象棋终极版，请执行以下步骤：

1. **克隆代码库**：
    ```bash
    git clone https://github.com/VoxDroid/Chess-Ultimate.git
    cd Chess-Ultimate
    ```

2. **在浏览器中打开**：
    ```bash
    open index.html
    ```
