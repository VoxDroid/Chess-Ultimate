## 🚀 Getting Started

To get started with Chess Ultimate locally:

1. **Clone the repo**:
    ```bash
    git clone https://github.com/VoxDroid/Chess-Ultimate.git
    cd Chess-Ultimate
    ```

2. **Open in your browser**:
    ```bash
    open index.html
    ```
