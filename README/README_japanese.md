## 🚀 始め方

チェスアルティメットをローカルで始めるためには:

1. **リポジトリをクローンする**:
    ```bash
    git clone https://github.com/VoxDroid/Chess-Ultimate.git
    cd Chess-Ultimate
    ```

2. **ブラウザで開く**:
    ```bash
    open index.html
    ```
